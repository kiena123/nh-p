- Bước 1 : " composer install "

- Bước 2 : " npm install "

- Bước 3 : " npm run dev "

- Bước 4 : " cp .env.example .env " Sửa lại DB_DATABASE và tạo database có tên trùng với giá trị của DB_DATABASE

- Bước 5 : " php artisan key:generate " để tại APP_KEY

- Bước 6 : " php artisan serve " để chạy server
